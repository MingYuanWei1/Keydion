PaperQuery项目是建立一个论文查询的网站项目。这个项目应该包括以下几个部分：
1.登陆页面，包括用户名，密码。用户数据存储采用cvs结构，包括用户名，密码，登记日期，过期时间，权限包括（1-只读，2-读和上传，3-读，上传，删除)三个等级。
2.查询页面（3个等级均可见），用户输入关键字，程序在给出的论文（PDF文件）中查询，给出符合的论文名称（论文题目或者内容里包括关键字里的），如果内容数量较多，给出前5个符合的。
3.上传页面（2，3等级可见），点击之后可以上传论文（PDF格式）。
4.删除页面（3级可见），给出论文列表（可以是全部，也可以是查询结果）可以点击删除，要确认2次以上。
5.网站是一个以英语为主的国际中学，所以网站风格应该贴切，符合国际学校的整体风格。

创建虚拟环境：py -3 -m venv .venv
激活：.\.venv\Scripts\Activate.ps1
安装依赖：python -m pip install -U pip
pip install -r requirements.txt
运行：.\start_local.ps1 -Secret "dev-secret-key" -DataDir ".\data" -UploadDir ".\assets\uploads" -Venv ".venv"
退出虚拟环境：deactivate

修改后编译语言模块：python tools/compile_translations.py
手动清理用户占用：data/active_sessions.json
